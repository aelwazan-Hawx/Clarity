# Finance Tracker App

A complete personal finance management application with dual currency support (SAR & EGP).

## Features

- ✅ Dashboard with income/expense overview
- ✅ Transaction management (add, edit, delete)
- ✅ Budget tracking with visual progress bars
- ✅ Category management
- ✅ Payment method tracking
- ✅ Opening balance management
- ✅ Dual currency support (SAR & EGP)
- ✅ Transaction filtering and search

## Tech Stack

**Frontend:**
- React 18
- Tailwind CSS
- Lucide React Icons

**Backend:**
- Node.js
- Express
- PostgreSQL
- JWT Authentication

## Setup Instructions

### Prerequisites
- Node.js 16+ installed
- PostgreSQL database (or Supabase account)

### Local Development

1. Clone repository
2. Setup frontend (see frontend/README.md)
3. Setup backend (see backend/README.md)

### Deployment

- Frontend: Vercel
- Backend: Render
- Database: Supabase
